## 🌟 Overview

**Aeva** is a modern, high-performance Discord music bot built for a smooth and premium user experience.  
Designed with clean UI, fast response time, and stable playback, Aeva makes music control effortless inside Discord servers.

---

## 🚀 Key Features

- 🎵 **High Quality Audio** — Smooth and stable playback experience  
- ⚡ **Fast Response** — Instant command execution  
- 🎛️ **Advanced Controls** — Play, Pause, Skip, Queue, Loop  
- 🎨 **Modern UI** — Clean and minimal embed design  
- 📂 **Multi-Source Support** — YouTube, Spotify, SoundCloud  
- 🔄 **Queue System** — Manage tracks easily  
- 🟢 **Stable Uptime** — Optimized performance  

---

## 🎮 Commands

| Command | Description |
|--------|------------|
| `/play` | Play music |
| `/skip` | Skip current track |
| `/pause` | Pause music |
| `/resume` | Resume playback |
| `/queue` | Show queue |
| `/stop` | Stop music |
